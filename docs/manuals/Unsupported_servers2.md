Support for a server doesn\'t come from nowhere - if your server is supported then that\'s because someone developed support for that server. Likewise, if your server is not supported, then that means someone will have to develop support for that server. If you have programming skills, please help us with writing support for your favorite server! Just post a message at [Developers Corner](http://forums.openkore.com/viewforum.php?f=36). Thank you.
